# SmartFileMan 📂✨

[English](README.md) | 简体中文

**SmartFileMan** 是一个基于 .NET 10 和 MAUI 构建的模块化智能文件管理系统。它超越了简单的规则分类，通过“竞价流水线”架构，允许不同插件根据文件元数据和上下文关联，自动、智能地整理你的磁盘空间。

## 🌟 核心特性

*   **智能竞价流水线**: 插件不再只是被动过滤，而是根据文件特征主动“出价”。系统通过得分算法自动仲裁最佳归档路径。
*   **全景分析**: 支持批处理模式。在整理前，插件会先行扫描整个文件夹（Phase 0），识别文件间的隐含关系（如：游戏存档与其预览截图、多卷压缩包）。
*   **增量处理与分身识别**: 内置 `FileTracker` 引擎。通过哈希值和元数据对比，自动跳过未变更的文件，即使文件被重命名也能精准追踪。
*   **高度模块化**: 插件完全隔离，支持**热插拔**。你可以根据需要随时添加图片识别、发票归档或音乐整理插件。
*   **开发者友好**: 提供 RSA 插件签名系统、内置 LiteDB 隔离存储、实时日志控制台和数据库 SQL 检查器。
*   **跨平台基础**: 基于 .NET MAUI 构建，原生支持多语言（中英双语）界面与文档。

## 🏗️ 项目架构

*   `SmartFileMan.Contracts`: 核心接口定义，定义了插件与系统的交互标准。
*   `SmartFileMan.Sdk`: 开发者工具包，包含 `PluginBase`、`SafeContext` 和常用的文件操作命令。
*   `SmartFileMan.Core`: 系统的核心大脑，负责插件加载、文件监控、竞价仲裁和调度。
*   `SmartFileMan.App`: 基于 .NET MAUI 的桌面级应用，提供流畅的 UI 交互和可视化数据管理。
*   `SmartFileMan.Signer`: 安全签名工具，确保加载的插件安全可靠。

## 🚀 快速开始

### 用户
1. 下载并运行 `SmartFileMan.App`。
2. 在“设置”中配置你要监控的文件夹。
3. 放入文件，观察系统自动分类。

### 开发者
阅读我们的 [开发者指南 (中文)](DEVELOPER_GUIDE_ZH.md) 开始编写你的第一个智能整理插件！

## 🛡️ 安全性

系统默认仅加载拥有合法数字签名的插件。如需开发和测试，请在“设置”中开启“开发者模式”。
